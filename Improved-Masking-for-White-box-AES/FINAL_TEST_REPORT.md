# 白盒AES算法测试与覆盖率分析报告

## 项目概述

本报告详细描述了对白盒AES算法实现的全面测试和代码覆盖率分析。该算法基于改进的掩码方案，用于抵御白盒密码学中的差分计算分析攻击。

我们开发了一个全面的测试套件，包含以下测试模块：

- **单元测试** (`test/unit_test.c`): 9个单元测试，覆盖每个函数的分支
- **边界测试** (`test/boundary_test.c`): 10个边界条件测试
- **稳定性测试** (`test/stability_test.c`): 10个稳定性和性能测试
- **WBMatrix测试** (`test/wbmatrix_test.c`): 矩阵库专项测试
- **覆盖率完善测试** (`test/quick_coverage_test.c`): 针对性覆盖率提升

### 2. 测试自动化
- **覆盖率测试脚本** (`run_coverage_test.sh`): 自动化测试执行
